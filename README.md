# 📌 Project Summary

This project is an **RPA (Robotic Process Automation)** solution developed using **UiPath Studio** to automate the complete attendance management process.

Instead of manually checking attendance records, the bot imports attendance data from Excel, calculates each student's attendance percentage, identifies students with low attendance, generates an attendance report in PDF format, and automatically sends email notifications.

The automation reduces manual work, improves accuracy, saves time, and ensures a faster attendance reporting process.

---

# 🚀 Features

- 📄 Reads attendance details from Excel
- 🤖 Processes student attendance automatically
- 📊 Calculates attendance percentage
- ⚠️ Identifies low attendance students
- 📑 Generates attendance reports
- 📃 Converts reports into PDF
- 📧 Sends reports through Email
- ✅ Eliminates manual attendance processing
